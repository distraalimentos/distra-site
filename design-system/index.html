Design System

Sistema de design da Distra Alimentos. Tokens, componentes e diretrizes compartilhados entre o site institucional, o portal de vendedores e demais projetos da marca.

Como usar

Para aplicar o design system em qualquer página nova:

<!-- No <head> -->
<link rel="stylesheet" href="/design-system/tokens.css">
<link rel="stylesheet" href="/design-system/components.css">

No Figma: instale o plugin Tokens Studio for Figma, vá em Settings → Import e carregue o arquivo tokens.json. Todas as cores, tipografia e espaçamentos aparecem como estilos prontos.

Regra de ouro: nunca use valores hex ou tamanhos soltos. Sempre referencie um token. Se precisar de um valor novo, adicione primeiro ao tokens.css.
